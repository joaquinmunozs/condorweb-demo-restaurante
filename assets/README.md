# 🎨 Logo del restaurante

El sitio ya trae un **logo vectorial** (escudo dorado con olla humeante) que funciona
como logo real. Si generas un logo propio (p. ej. con ChatGPT), reemplázalo así:

## Para usar tu propio logo
1. Genera el logo en **PNG con fondo transparente** (cuadrado, alta resolución).
2. Guárdalo en esta carpeta exactamente como: **`assets/logo.png`**
3. Listo: el sitio lo detecta solo y reemplaza al logo vectorial en la barra superior.

> El escudo grande (cabecera de la carta y hero del inicio) seguirá siendo vectorial.
> Si quieres usar tu imagen también ahí, avísame y lo conecto.

## Prompt usado para generar el logo (ChatGPT / DALL·E)
Ver el archivo `LOGO_PROMPT.md` en la raíz del proyecto.

## Paleta de marca
- Dorado champagne: `#c8a96e`
- Crema: `#fdf6e3`
- Rojo vino: `#8b1a1a`
- Fondo oscuro: `#16120f`
